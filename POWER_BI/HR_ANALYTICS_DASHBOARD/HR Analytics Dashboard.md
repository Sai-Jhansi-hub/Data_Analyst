### **HR Analytics Dashboard**



#### **Problem Statement:**

You are working as an HR Analyst, and your task is to study employee attrition and workforce trends across various departments, age groups, education fields, and salary ranges.



In this project, you need to build a Power BI dashboard that visualizes the attrition data, highlights KPIs, and gives clear insights into employee turnover and satisfaction levels. (Create a Power BI dashboard to understand workforce attrition trends using the given HR dataset.)



This project will test your skills in data importing, cleaning, modeling, DAX creation, and dashboard visualization.



🔗 Dataset:



Download the dataset from the following link:



→ HR Analytics Dataset (Excel file): Link



(Contains: HR Data.xlsx)



Your Task:

You must design and develop an interactive dashboard that meets the following requirements:





#### Task 1: Data Preparation



Import the HR\_data Excel file into Power BI.

Clean the dataset (rename columns if needed, correct data types like numeric and categorical fields).

Handle nulls or duplicates.

Rename fields for better readability (example: "MonthlyIncome" instead of "Monthly\_Income").



#### Task 2: KPI Creation :



Create KPI Cards for :



Total Employees (count of Employee Numbers)

Attrition Rate (Measure: Attrition Rate = (Employees Left / Total Employees) × 100)

Average Monthly Income (average of Monthly Income column)

Format all KPI cards with consistent font size, background, and text color.

Arrange the KPI cards neatly at the top of the dashboard for quick highlights.



#### Task 3: Visualizations :

Design a professional dashboard that includes:



Donut Chart – Attrition Rate by Education Field

Clustered Column Chart – Attrition by Age Group

Horizontal Bar Chart – Attrition by Salary Slab

Matrix Visual – Job Role vs Job Satisfaction Level

Line Chart – Years at Company vs Attrition Count

Column Chart – Department vs Number of Employees Left

Slicers – Add filters for: Gender , Department

Dashboard Title: Add a bold title "HR Analytics Dashboard" at the top using a text box.





Dashboard Design Guidelines :

Use a clean, modern design with a light background and dark text for professional appearance.

Maintain consistent font styles and sizing across all visuals.

Use spacing, borders, and shadows to make visuals distinct.

Use Format Painter tool for styling consistency across charts and cards.

Turn on data labels where required and align all visuals properly for better readability.
